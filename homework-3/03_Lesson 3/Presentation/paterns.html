Зачем нужны массивы ?
Положим, вы решили написать программу для учета всех видов динозавров, которые вам известны. Вы можете создать для каждого вида отдельную переменную:
1 var dinosaurik1 = "Тираннозавр";
2 var dinosaurik2 = "Велоцираптор";
3 var dinosaurik3 = "Стегозавр";
4 var dinosaurik4 = "Трицератопс";
5 var dinosaurik5 = "Брахиозавр";
6 var dinosaurik6 = "Птеранодон";
7 var dinosaurik7 = "Апатозавр";
8 var dinosaurik8 = "Диплодок";
9 var dinosaurik9 = "Компсогнат";
Однако пользоваться этим списком не слишком удобно — у вас есть девять переменных там, где можно обойтись лишь одной.


Массивы
Массив – это упорядоченная коллекция значений. Значения в массиве называются элементами, и каждый элемент характеризуется числовой позицией в массиве, которая называется индексом. Массивы в языке JavaScript являются нети­пизированными: элементы массива могут иметь любой тип, причем разные элементы одного и того же массива могут иметь разные типы.
Создание массива
1// Способ 1
2var name = [ ]
3// Способ 2
4var name = new Array()
Фактически для задания пустого массива достаточно лишь пары квадратных скобок
Как будет выглядеть массив с драконами ?
Чтобы создать массив со значениями, нужно перечислить эти значения внутри квадратных скобок, разделяя их запятыми. Отдельные значения, хранящиеся в массиве, называют элементами. В данном примере все элементы будут строковыми
1var dinosaurs = ["Тираннозавр", "Велоцираптор",
2"Стегозавр","Трицератопс", "Брахиозавр",
3"Птеранодон", "Апатозавр","Диплодок",
4"Компсогнат"];
Доступ к элементам по имени
Чтобы получить доступ к элементам массива, используйте квадратные скобки с индексом нужного вам элемента, как в этом примере:
1dinosaurs[0];
2"Тираннозавр"
3dinosaurs[3];
4"Трицератопс"
Индекс — это номер элемента, в котором хранится значение.
Многомерные массивы
JavaScript не поддерживает «настоящие» многомерные массивы, но позволяет неплохо имитировать их при помощи массивов из массивов. Для доступа к эле-менту данных в массиве массивов достаточно дважды использовать оператор []. Например, предположим, что переменная matrix – это массив массивов чисел. Ка-ж дый элемент matrix[x] – это массив чисел. Для доступа копределенному числу в массиве можно использовать выражение matrix[x][y]. Ниже приводится кон-кретный пример, где двумерный мас сивиспользуется в качестве таблицы умно-жения:
